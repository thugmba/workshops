2. Chart Anatomy & Type Selection

Core Chart Anatomy

Axes
The horizontal and vertical reference lines (x/y axes).
Labels
Explanatory text identifying axes and data categories.
Ticks
Grid markers and numeric values showing scaling.
Legends
Key explaining visual markers, colors, or line styles.

Chart Type Selection

  • Line Chart: Continuous data changes over time / trends.
  • Bar Chart: Comparing sizes of discrete categorical items.
  • Pie Chart: Part-to-whole proportions (small groups).
  • Box Plot: Statistical distribution and outliers.
  • Scatter Plot: Relationships between continuous variables.

5. Lab 1: Marketing Channel Distribution

Goal

Visualize how prospective international students first hear about the GMBA program.

Visual Principle

Sorting & Label Clutter: Always sort categorical charts by frequency (descending). Rotate labels by 45° to prevent overlaps.

Vibe Coding Tip: Tell the agent to clean white spaces, aggregate using value counts, and apply a specific rotation factor to the x-tick labels.

5. Lab 2: Importance vs Sufficiency Gap

Goal

Compare perceived importance of homepage info (Q3) vs current sufficiency (Q4).

Visual Principle

Grouped Charts: Side-by-side bars highlight discrepancies. Map Likert scales (text) to numeric integers (1-5) before averaging.

Vibe Coding Tip: Instruct the agent to map textual Likert scales to numeric integers using pandas mapping.

5. Lab 3: Usability Barriers Heatmap

Goal

Chart specific navigation and usability barriers (Q6) students face.

Visual Principle

Matrix Data: Melt wide tables into long format. Use sequential color gradients (heatmaps) to draw attention to pain points.

Vibe Coding Tip: Prompt the agent to use pandas melt function to reshape the columns before plotting.

5. Lab 4: Satisfaction Dashboard

Goal

Assemble a unified HTML dashboard: Overall satisfaction (Q10) + Preferred formats (Q9).

Visual Principle

Hierarchy: KPIs top-left (F-pattern). Group details to right/bottom. Ensure color consistency.

Vibe Coding Tip: Direct the agent to generate a combined Plotly grid layout or a simple HTML/CSS template.
