Core Chart Anatomy Foundations

Key Component Definitions

  • Title — States the central message and comparison of the chart.
  • Axes (X-Axis & Y-Axis) — Vertical and horizontal scales holding numbers or categories.
  • Axis Labels — Identify what metric or category is displayed.
  • Ticks & Gridlines — Guide references showing specific measurement levels.
  • Legend — Key mapping visual colors/shapes to categorical series.

Chart Type Selection Decisions

Line Chart

Best for trends over time (e.g., changes over weeks or months).

Bar Chart

Best for comparisons between discrete categorical groups.

Pie / Donut

Best for parts of a whole. Limit categories to 5 for readability.

Box Plot

Best for showing statistical distributions and outliers.

Scatter Plot

Best for correlation and relationships between 2 variables.

The Vibe Coding Loop Concept

Instead of manually coding charts, you instruct the AI using natural English. You review the output chart, identify visual flaws, and prompt the agent iteratively until the chart is ready.

Creating Custom Agent Skills Skills

The Custom Skill Concept

A Custom Skill is a markdown document (SKILL.md) outlining prompt guidelines, color tokens, and layout rules.

  • Why build skills? Prevents repeating styling instructions (e.g., font family, margins, color palettes) in every new conversation.
  • Autonomous load: The agent detects the skill based on your prompt intent and applies rules automatically.

Directory Locations

Workspace level (local):

.agents/skills/google_antigravity_style/SKILL.md

User level (global):

/home/sun/.gemini/config/skills/google_antigravity_style/SKILL.md

Structure custom skills with YAML metadata (name, description) at the very top of your SKILL.md file.

Prompts for Data Viz AI Best Practices

Three-Part Prompt Formula

Maximize visual consistency by structuring your agent prompts into three key sections:

  1. Goal (What to plot): Specify chart types, metrics, and aggregation rules.
  2. Context (Data structure): Mention input variables, thresholds, and mappings.
  3. Style (Aesthetics): Highlight palette colors, text wrapping, and borders.

Structured Prompt Template

"Act as a clean, modern dashboard designer.
Using 'homepage_ux.csv', plot a seaborn grouped bar chart.
1. Compare 'Importance' vs 'Sufficiency' mean scores.
2. Abbreviate long category names inside square brackets.
3. Apply deep blue for Importance, soft teal for Sufficiency.
4. Rotate X-tick labels by 45 degrees.
5. Apply sns.despine() to clean the right and top borders."
