# Innovation Analytics Lab Workshops

Welcome to the **Innovation Analytics Lab** workshop directory! This repository hosts the public training materials, interactive guides, slide decks, and datasets designed for business and non-engineering students at Tunghai University Global MBA.

Please visit the central workshop portal below to access all resources in a unified interface:

**[Launch the Workshop Portal Hub](https://thugmba.github.io/workshops/)**

## Quick Links to Workshop Resources

## How to Get Started

1. Open the **[Workshop Portal Hub](https://thugmba.github.io/workshops/)** in your web browser.
2. Select the workshop you want to practice.
3. Open the corresponding interactive slide deck to follow the lecture.
4. Open the practice handbook to read the step-by-step instructions.
5. Download the necessary datasets (such as `homepage_ux.csv` for the Data Viz workshop) to proceed with the hands-on labs.
