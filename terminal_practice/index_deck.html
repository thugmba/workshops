Introduction to the command line interface (CLI): controlling your computer via text commands instead of clicking icons.

Platforms:
  • macOS / Linux (Unix)
  • Windows
Importance:

Foundational skill for business and non-engineering students in AI and Machine Learning courses.

2. Essential Terminology (1/2)

Directory / Folder
Location in a file system used to store and organize files.
Parent Directory
A directory that contains one or more subdirectories.
Child Directory
A directory located inside another directory.
Current Working Directory
The directory you are currently "inside" in the terminal.
Home Directory
Default personal directory assigned to a user (~ on Unix).

2. Essential Terminology (2/2)

Root Directory
The top-most directory in a file system (/ on Unix, C:\ on Windows).
Project Root Directory
The top-most directory specific to a particular project.
Absolute Path
The complete path starting from the root directory.
Relative Path
The path relative to your current working directory.
Terminal / Shell
The text-based application interface used to execute commands.

3. How to Open the Terminal

macOS Cmd + Space → type Terminal → Enter
Windows Win key → type cmd → Enter
Linux Ctrl + Alt + T (Ubuntu/Common)

6. Student Evaluation (1/2)

Scenario: Set up a research workspace climate_research_[name]

Required Structure:
  • src_[name], output_[name], references_[name], doc_[name]
  • Nested: paper_[name]/drafts_[name]
  • File: doc_[name]/todo.txt (with name and tasks)
  • File: references_[name]/sources.txt

6. Student Evaluation (2/2)

Refactor: Rename doc_[name] → documentation_[name] and sources.txt → citations.txt.
Cleanup: Delete output_[name] directory.

Verification Command:

macOS/Linux: date && tree && cat documentation_[name]/todo.txt
Windows: echo %date% %time% && tree /F && type documentation_[name]\todo.txt

6.4 Evaluation Criteria

  • Terminal Launch: Can open the command-line application correctly.
  • Navigation: Can move between nested directories (cd, cd ..).
  • Pathing: Understands custom naming and nested directory creation.
  • File Management: Able to create, rename, and delete files/directories.
  • Verification: Able to generate the required report screenshot.
